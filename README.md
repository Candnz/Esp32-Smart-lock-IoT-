# Esp32-Smart-lock-IoT-
Bu proje AKTİF olarak geliştiriliyor.

Smart personel parking lock controlled by mobile application.(Kişisel kullanım için akıllı park dubası)
## Duba telefondan kontrol edilebilecek şekilde tasarlandı araç park edilmek istenince uygulamadan kapat tuşuna basılacak ve duba inecek tersi işlem için de aç tuşuna basılacak. 

✅ÖZELLİKLER:
- Esp32 ile Wi-fi/Bluetooth üzerinden komut alma 
- Limit switch ile doğru konumlanma
- Firebase ile mobil uygulama üzerinden gecikmesiz kontrol
  
**Donanım**
-Esp32
-Silecek Motoru
-BT7960 motor sürücü
-Limit switch

**Mobil Uygulama**
-MIT App Inventor
-Firebase Realtime Database
